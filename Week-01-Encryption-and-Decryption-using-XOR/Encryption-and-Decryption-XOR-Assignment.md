## Encryption/Decryption Using XOR

### Objective

Learn how to encrypt and decrypt messages using the XOR logical operator.

### Prerequisites

- Review the lecture material before starting the lab.

### Task

**Please note:** No programming is required for this activity. Use paper and pencil to solve the following tasks.

#### Encrypting the Message

1. Select a secret one-word English message.
2. Convert the message into binary using the ASCII representation of each character.
3. Create a secret one-word key of the same length as the message and convert it into binary.
4. Apply the XOR operation between the message and the key.
5. Convert the encrypted binary result into hexadecimal form.

#### Decrypting the Message

1. Use the encrypted message and the secret key to perform XOR.
2. Verify that the decrypted message matches the original plaintext.
3. Explain how XOR encryption can be performed when the plaintext and key have different lengths.

### What to Submit

3. Encryption process and calculations (4 marks).
4. Decryption process and calculations (4 marks).
